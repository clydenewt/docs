# Neutron Developer Tutorials

The following tutorials are designed to help you learn how to build on Neutron, from basic CosmWasm smart contracts to advanced cross-chain applications. Each tutorial builds on the previous one, creating a comprehensive learning path.

## Learning Path

Our tutorials follow a progression from basic to advanced concepts. For beginners, we recommend starting with the <strong>Introduction to CosmWasm</strong> and working through the tutorials in order.

## Getting Started

Before starting the tutorials, we recommend setting up your development environment:

1. **Install Prerequisites**:
   - [Rust](https://www.rust-lang.org/tools/install)
   - [Go](https://go.dev/doc/install) (for running a local node)
   - [Docker](https://docs.docker.com/get-docker/) (for some tutorials)

2. **Clone the Example Repository**:
   ```bash
   git clone https://github.com/neutron-org/neutron-sdk.git
   cd neutron-sdk
   ```

3. **Set Up a Local Development Environment**:
   Follow our [quickstart guide](/developers/quickstart) to set up a local Neutron development environment.
