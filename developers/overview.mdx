## About Cosmopark

Cosmopark is our recommended local development environment that sets up multiple interconnected components:

<div className="p-4 border-l-4 border-primary bg-primary/5 rounded my-6">
  <ul className="mt-2">
    <li><strong>Neutron node</strong>: A fully-functional local Neutron blockchain</li>
    <li><strong>Gaia node</strong>: A Cosmos Hub node connected via IBC</li>
    <li><strong>Hermes IBC Relayer</strong>: Maintains IBC connections and transfers packets</li>
    <li><strong>Neutron ICQ Relayer</strong>: Facilitates cross-chain state queries</li>
  </ul>
</div>

This environment is ideal for testing Interchain Accounts (ICA), Interchain Queries (ICQ), and other Neutron-specific features. [Learn more about Cosmopark →](/developers/tutorials/onboarding/cosmopark)
