## For Users

<Steps>
  <Step title="Access the Neutron App">
    Visit [app.neutron.org](https://app.neutron.org/) to access the Neutron DeFi ecosystem.
  </Step>
  <Step title="Connect Your Wallet">
    Connect a compatible wallet to the Neutron app by clicking "Connect Wallet" and selecting your preferred wallet provider.
    <Note>
      We recommend checking out wallets like [Keplr](https://www.keplr.app/get) and [Leap](https://www.leapwallet.io/download) for the best experience.
    </Note>
  </Step>
  <Step title="Explore Neutron's DeFi Ecosystem">
    Neutron offers a comprehensive DeFi ecosystem with multiple ways to participate:
    
    - **Supervaults**: Automatically compound your yields across multiple strategies
    - **Liquid Staking**: Stake NTRN and receive liquid staking tokens you can use throughout the ecosystem
    - **Lending & Borrowing**: Access capital markets for a variety of assets
    - **Trading**: Swap assets easily with minimal slippage 
    - **Governance**: Participate in shaping Neutron's future
    
    For more information on Neutron's DeFi features, check out our [DeFi Guide](/defi/overview).
  </Step>
</Steps>

## For Developers

<Steps>
  <Step title="Set Up Development Environment">
    Install the required dependencies:
    
    ```bash
    # Install Rust
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    
    # Install Go
    wget https://go.dev/dl/go1.23.1.linux-amd64.tar.gz
    sudo tar -C /usr/local -xzf go1.23.1.linux-amd64.tar.gz
    ```
  </Step>
  <Step title="Clone Neutron Repository">
    ```bash
    git clone https://github.com/neutron-org/neutron.git
    cd neutron
    ```
  </Step>
  <Step title="Build and Install Neutron">
    ```bash
    # Build the neutrond binary
    make install
    
    # Add to PATH (add this to your .bashrc or .zshrc for persistence)
    export PATH=$PATH:$(go env GOPATH)/bin
    
    # Verify installation
    neutrond version
    ```
  </Step>
  <Step title="Start a CosmWasm Project (Optional)">
    ```bash
    # Generate a new CosmWasm project using the template
    cargo generate --git https://github.com/CosmWasm/cw-template.git --name neutron-ica-example
    cd neutron-ica-example
    
    # Build the contract
    cargo wasm
    ```
  </Step>
  <Step title="Explore Developer Documentation">
    Visit the [Developer Overview](/developers/overview) for detailed guides on:
    - Building smart contracts
    - Using Neutron's modules
    - Setting up a local testnet
  </Step>
</Steps>

## For Validators

<Steps>
  <Step title="Prepare System Requirements">
    Ensure your system meets the minimum requirements:
    
    - 4 CPU cores
    - 16GB RAM
    - 500GB SSD
    - Ubuntu 20.04 or later
  </Step>
  <Step title="Set Up a Neutron Node">
    Follow our [Running a Node](/validators/running-a-node) guide to set up your Neutron full node.
  </Step>
  <Step title="Become a Validator">
    Once your node is synced, visit [Becoming a Validator](/validators/becoming-a-validator) to learn how to:
    - Create a validator
    - Set commission rates
    - Join the active set
  </Step>
</Steps>
