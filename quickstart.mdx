# Quickstart Guide

This guide will help you quickly get started with Neutron, whether you're a user, developer, or validator.

## For Users

<Steps>
  <Step title="Access Neutron App">
    Visit [app.neutron.org](https://app.neutron.org) to access the Neutron application.
  </Step>
  <Step title="Connect Wallet">
    Connect your Keplr, Leap, or other Cosmos-compatible wallet.
    
    <Note>
      If you don't have a wallet yet, we recommend installing [Keplr](https://www.keplr.app/) or [Leap Wallet](https://www.leapwallet.io/).
    </Note>
  </Step>
  <Step title="Explore DeFi Options">
    Once connected, you can:
    - Stake NTRN tokens
    - Use Supervaults for automated yield strategies
    - Trade on the DEX
    - Participate in governance
  </Step>
</Steps>

## For Developers

<Steps>
  <Step title="Set Up Development Environment">
    Install the required dependencies:
    
    ```bash
    # Install Rust
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    
    # Install Go
    wget https://go.dev/dl/go1.19.linux-amd64.tar.gz
    sudo tar -C /usr/local -xzf go1.19.linux-amd64.tar.gz
    ```
  </Step>
  <Step title="Clone Neutron Repository">
    ```bash
    git clone https://github.com/neutron-org/neutron.git
    cd neutron
    ```
  </Step>
  <Step title="Explore Developer Documentation">
    Visit the [Developer Overview](/developers/overview) for detailed guides on:
    - Building smart contracts
    - Using Neutron's modules
    - Setting up a local testnet
  </Step>
</Steps>

## For Validators

<Steps>
  <Step title="Prepare System Requirements">
    Ensure your system meets the minimum requirements:
    
    - 4 CPU cores
    - 16GB RAM
    - 500GB SSD
    - Ubuntu 20.04 or later
  </Step>
  <Step title="Set Up a Neutron Node">
    Follow our [Running a Node](/validators/running-a-node) guide to set up your Neutron full node.
  </Step>
  <Step title="Become a Validator">
    Once your node is synced, visit [Becoming a Validator](/validators/becoming-a-validator) to learn how to:
    - Create a validator
    - Set commission rates
    - Join the active set
  </Step>
</Steps>
